The giveaways addon are designed to help you create and manage giveaways in your server. You can create conditional giveaways, set the number of winners, and more.

[Purchase](https://builtbybit.com/resources/54939/)

## Commands
- `/giveaway create <prize> <time> [winners] [channel] [condition]`<br/>
  Create a giveaway. The giveaway will be created in the specified channel or the current channel if no channel is specified.
- `/giveaway end <message-id>`<br/>
  End a giveaway. The giveaway will be ended and the winners will be announced.
- `/giveaway reroll <message-id> [winners]`<br/>
  Reroll a giveaway. The giveaway will be rerolled and new winners will be announced.
  Also available as a context menu on the giveaway message.
