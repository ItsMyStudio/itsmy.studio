MCStatus is an ItsMyBot addon that adds placeholders to retrieve information from a Minecraft server anywhere. ItsMyBot placeholders can be used in any message, other addons, scripts, etc.

[Download](https://builtbybit.com/resources/31222/)

## Placeholders
- `%mcstatus_<server>_players%`<br/>
  The amount of players online in the server.
- `%mcstatus_<server>_max_players%`<br/>
  The maximum amount of players the server can have.
- `%mcstatus_<server>_motd%`<br/>
  The message of the day of the server.
- `%mcstatus_<server>_version%`<br/>
  The version of the server.
- `%mcstatus_<server>_host%`<br/>
  The host of the server.
- `%mcstatus_<server>_port%`<br/>
  The port of the server.
- `%mcstatus_<server>_status%`<br/>
  The status of the server. It can be either "online" or "offline".
