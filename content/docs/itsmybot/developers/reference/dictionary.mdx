This page is a glossary for the terms that come up most often in the ItsMyBot documentation.

It is meant to answer one question fast: "What does this word mean in the framework?"

## Core terms

<TypeTable
  type={{
    Addon: {

    },
    Manager: {

      type: 'Core Class',
    },
    Service: {
      description: 'A manager-owned runtime service such as the interaction service, event service, action service, or expansion service.',
      type: 'Core Class',
    },
    'Primary Guild': {
      description: 'The main guild configured in the core config. Many built-in features register and operate against this guild.',
      type: 'Runtime Term',
    },
    Resource: {
